学习Readme怎么写的。
===========================
> 在项目过程中有个良好的readme，可以很好的帮助队友了解项目规范和进度。知道现在开发中的一些问题和注意事项，也是互相记录的一个记录点。

* [参考文章](http://blog.csdn.net/guodongxiaren/article/details/23690801)

### 横线的表示方式
```
-----------               可以显示横线效果
===================       可以显示横线效果
***、---、___             可以显示横线效果
```
